# SunuLink

SunuLink est une plateforme de recrutement basée sur l’intelligence artificielle conçue pour connecter les talents aux opportunités professionnelles.

<img width="836" height="431" alt="sunulink" src="https://github.com/user-attachments/assets/25feba18-ad73-43db-93ec-d24b0cf28158" />



## Objectif

Le projet vise à améliorer le processus de recrutement en automatisant l’analyse des CV et en proposant des correspondances pertinentes entre candidats et offres d’emploi.

## Fonctionnalités principales

* Création de profil utilisateur
* Importation et analyse automatique de CV
* Extraction des compétences et expériences
* Structuration des données professionnelles
* Tableau de bord utilisateur
* Système de recommandation d’opportunités

## Technologies utilisées

* Python
* FastAPI
* Next.js
* Supabase
* NLP pour l’analyse de CV
